# Hosker's Almanack

## Getting Going

### Before You Begin

Ensure you have the following installed on your system:

* python3
* sqlite3
* sqlite-browser

### Running

Change directory to this one, and then run:

    python3 pdf_builder.py

You can then find the *Almanack* at `almanack.pdf`.
